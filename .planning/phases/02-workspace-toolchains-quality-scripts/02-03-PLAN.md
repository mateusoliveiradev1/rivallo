<threat_model>
## Trust Boundaries

| Boundary | Description |
|---|---|
| Root source/config files to quality tools | Tool configuration determines which files are inspected and whether warnings block. |
| Aggregate command to child commands | The aggregate propagates failures from tools that execute local repository code. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|---|---|---|---|---|
| T-02-06 | Tampering | Lint/typecheck include sets | mitigate | Target explicit existing root assets and make warnings blocking. |
| T-02-07 | Repudiation | Aggregate quality status | mitigate | Preserve the first child status and emit its atomic reproduction command. |
| T-02-08 | Denial of service | Cross-platform script execution | mitigate | Use Node process APIs rather than shell chaining. |
| T-02-SC | Tampering | Format checks | mitigate | Keep write-mode formatting out of aggregate validation. |
</threat_model>
