<threat_model>

## Trust Boundaries

| Boundary | Description |
|---|---|
| Tauri invoke response → TypeScript client | Generic IPC JSON is untrusted despite compile-time type parameters and must be decoded at runtime. |
| Legacy browser storage → import request | Local payloads may be corrupt, oversized, unknown-only, stale, or replayed under React StrictMode. |
| UI invoke request → host filesystem repository | The UI may submit malformed state but must never choose a path or gain file authority. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|---|---|---|---|---|
| T-061-04-T | Tampering | client decoder / Tauri DTO conversion | mitigate | Closed discriminants, finite bounds, exact IDs, Rust application validation, and malformed-response tests at both sides. |
| T-061-04-R | Repudiation | legacy retirement | mitigate | Exact source version/fingerprint durable receipt is required before any table field/key is retired. |
| T-061-04-D | Denial of Service | legacy/IPC payloads | mitigate | Bound bytes, strings, arrays, filter depth/count, and evidence before decode or repository mutation. |
| T-061-04-I | Information Disclosure | host errors and legacy payload | mitigate | Return typed reason codes and bounded identifiers; never return paths, raw quarantine bytes, or log full payloads. |
| T-061-04-E | Elevation of Privilege | repository path / future commands | mitigate | Host injects one fixed app-data path and exposes no identity, SQLite, query, cache, sync, network, or arbitrary-path command. |
</threat_model>
