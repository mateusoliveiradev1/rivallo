## Sampling Rate

- **After every frontend task commit:** Run `pnpm exec vitest run apps/desktop/src/matchday` plus the task's focused test file.
- **After every Rust task commit:** Run `pnpm rust:test` plus the task's focused crate test when available.
- **After every contract/schema task:** Run `node --test tooling-tests/phase-5-table-view-contract.test.mjs` plus focused engine tests.
- **After every plan wave:** Run `pnpm components:test && pnpm rust:test && pnpm rust:architecture && pnpm typecheck`.
- **Before `$gsd-verify-work`:** The full suite command above must be green.
- **Max feedback latency:** 60 seconds for task-local checks; browser/build gates run at wave and phase boundaries.
