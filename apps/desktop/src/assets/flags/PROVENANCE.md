# Local nationality flags

These four compact SVG assets were authored in-repository for Rivallo on 2026-07-15 from the public construction and colour conventions of the national flags they identify:

- `ar.svg` — Argentina;
- `br.svg` — Brazil;
- `pt.svg` — Portugal;
- `uy.svg` — Uruguay.

No third-party SVG path, emoji glyph, game asset, remote image URL, or flag package was copied or imported. National flags are public identifying symbols; this repository owns only its compact SVG markup and rendering treatment. The assets are contextual data and never replace the visible country code or accessible full country name.
