# ADR-0002 Tauri + React + Rust

**Status:** Accepted · **Date:** 2026-07-13

**Context:** desktop requires responsiveness and Rust alignment. **Decision:** Tauri 2 shell, React/TypeScript/Vite UI, Rust services. **Alternatives:** Electron; native-only UI. **Consequences:** compact desktop delivery and web UI velocity. **Risks:** bridge boundaries. **Review:** platform capability or performance failure.
