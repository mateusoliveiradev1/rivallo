# ADR-0009 Impeccable and design system

**Status:** Accepted · **Date:** 2026-07-13

**Context:** dense desktop UI needs intentional visual QA. **Decision:** semantic token source, UI Lab, approved briefs, Impeccable reports/detect CI. **Alternatives:** default component styling; Storybook initially. **Consequences:** deliberate design gates. **Risks:** process overhead. **Review:** UI workflow friction.
